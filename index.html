생성형 AI란?

대규모 데이터로 학습한 모델이 프롬프트를 받아 텍스트, 이미지, 오디오, 코드 등 새로운 콘텐츠를 생성하는 기술입니다.

핵심 모델

  • LLM(대형 언어 모델) — 텍스트 생성
  • Diffusion/Transformer 기반 이미지 모델 — 이미지 합성
  • 음성 합성 모델 — TTS
